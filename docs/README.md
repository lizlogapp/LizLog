# APP 文件索引

| 檔案 | 用途 | 同步限制 |
| --- | --- | --- |
| `project-spec.txt` | APP 功能與產品原始規格。 | 可整理後同步。 |
| `persona.txt` | 使用者人物誌。 | 移除個資後可同步。 |
| `palette.md` | 色彩與主題變體說明。 | 可同步。 |
| `user-research-summary.md` | 匿名研究摘要與待補欄位。 | 可同步。 |

原始研究資料若包含姓名、email、電話或其他可識別資訊，僅可放在
`09_archive/private/`，不得提交到 Git 或同步至 Notion。
